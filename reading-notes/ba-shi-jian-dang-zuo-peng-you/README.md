## 第 0 章 困境
### 问题

任务在 Deadline 之前无法完成

| 看起来的问题原因 | 看上去的解决方案 |
|:--- |:--- |
| 没有按时开始执行任务 | 按时开始 |
| 错误估计完成任务所需时间 | 正确估算 |
| 在任务的执行过程中出现了差错 | 不要出差错 |

### 慌乱
既勤奋又懒惰

### 解决
- 问题出在我们自己身上
- 我们无法管理时间. 我们真正能够管理的, 只有我们自己.
